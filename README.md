# HealthBeauty AI – AI-Powered Health & Beauty Platform

A full-stack web application that combines dental clinic management with an AI-driven cosmetics e-commerce store.

## Problem Statement
- Patients struggle to get instant symptom analysis & book trusted doctors  
- Beauty shoppers receive generic product recommendations → wrong purchases & skin issues  
- Small clinics & salons lack an affordable, all-in-one digital solution

## Solution & Impact
- AI Symptom Checker + Skin-Tone Analyzer (Hugging Face models) → personalized advice in <3 seconds  
- Multi-panel system: Patient | Doctor | Admin | E-commerce User  
- Doctors can manage appointments, prescriptions & patient records  
- Hyper-personalized product recommendations → higher satisfaction & sales

## Unique Features
- Hybrid database (MySQL + MongoDB)  
- JWT + role-based secure authentication  
- Razorpay payment integration (test mode)  
- Scalable & Docker-ready architecture  
- Complete SDLC documentation

## Tech Stack
- React + Vite + Tailwind  
- Node.js + Express  
- MySQL + MongoDB  
- Hugging Face Inference API  
- Docker + Vercel deployment
